# Johns-Hopkins-Epic-Billing
Healthcare billing analysis project using Excel to explore monthly revenue trends at Johns Hopkins in 2024 using pivot tables and data visualization.

PROJECT OVERVIEW

Analyze monthly billing totals to identify trends in healthcare revenue at Johns Hopkins in 2024.

---------------------------------------------

DATASET

Healthcare billing dataset from Kaggle
URL: https://www.kaggle.com/datasets/nirmalptl/john-hopkins-healthcare-dataset

---------------------------------------------

TOOLS

* Excel
* Pivot Tables
* Column Charts
* Data Cleaning and Transformation

---------------------------------------------

PROCESS

* Imported CSV dataset from Kaggle

* Reviewed imported CSV data, verified column data types, formatted date fields for monthly analysis, and prepared the dataset for aggregation using Pivot Tables

* Created a Pivot Table to aggregate billing amounts by month and calculate total billed revenue over time

* Generated visualization (Column Chart) to show Total Billed Per Month

---------------------------------------------

FINDINGS

Monthly billing ranged from approximately $813K to $927K in 2024.

August recorded the highest revenue at $926,734.29, while February recorded the lowest revenue at $813,835.72.

There is no clear, sustained upward or downward trend throughout 2024, despite month-to-month fluctuations. This suggests relatively stable demand for healthcare services at Johns Hopkins, with month-to-month variation potentially influenced by seasonal or operational differences.

---------------------------------------------

LIMITATIONS

* No segmentation - Analysis focused on monthly billing totals and did not examine other groups, such as medical procedure (CPT_Code) or service area locations (SERV_AREA_ID). Therefore, insights into the underlying drivers of revenue fluctuations were limited.

* No external context - Analysis did not include holidays, flu seasons, staffing changes, and other real-life variables that may have influenced revenue.

* No multi-year comparison - Analysis was limited to 2024 data, preventing assessment of long-term trends at Johns Hopkins.
